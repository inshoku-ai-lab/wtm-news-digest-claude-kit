# WTM News Digest — Claude 版 複製キット

Telegram チャネル「WeTheMedia（WTM）」の公開投稿から、ニュース編集用のデータセットと解説記事を Claude を使って作成するワークフローの、そのままインストール可能な複製キットです。

同じ作業を、以前は NotebookLM ベースで公開したことがあります。本キットはその Claude 版であり、作業量を大幅に減らしつつ、出力の品質を維持することを目的としています。

---

## このキットで再現できるもの

1 週間分の Telegram エクスポート（英語版 + 日本語訳版）を投入すると、以下の 3 つの成果物が生成されます。

- **統合データセット**：各投稿の投稿日時、参照リンク、リアクション分析、テキストログ、ビジュアル解析（画像 OCR・動画字幕）、さらに追加分析 4 項目（反応レベル・要約・投稿タイトル・投稿ジャンル）を備えた Markdown。
- **メインニュース解説記事**：全投稿を横断集計し、複数の投稿が言及するトピック上位 5 件を選定。各トピックにつき 600〜800 字の中道保守トーンの解説記事、実在検証済みの関連外部記事 3 件、関連投稿リスト。
- **ジャンル別サブリスト**：メインニュースで使用した投稿を除外した残りを、10 種の固定ジャンルに分類し反応数順に並べたリスト。

---

## 動作環境

- **推奨**：Claude Pro または Claude Max（Projects と Skills が快適に使えます）
- **無料プランでも技術的には可能**（Projects は 5 個まで作成可、Skills も個人アップロード可）。ただし処理速度・メッセージ上限・コンテキスト管理の面で Pro 以上を強く推奨します。

## セットアップ手順（所要時間：5〜10 分）

### ステップ 1：新しい Claude Project を作成

Claude のサイドバーから「Projects」を開き、右上の「新規プロジェクト」を押します。名前は任意（例：`WTM News Digest`）で構いません。

### ステップ 2：プロジェクト指示を貼り付ける

作成したプロジェクトの「プロジェクト指示（Custom instructions）」欄に、本レポジトリの [`project-instructions.md`](./project-instructions.md) の中身をそのままコピーして貼り付けます。

### ステップ 3：スキルを登録する

Claude の「Customize → Skills」画面で、本レポジトリの [`skills/wtm-news-digest/`](./skills/wtm-news-digest/) フォルダを ZIP 圧縮したものをアップロードします。登録後、スキル一覧でトグルを ON にします。コード実行機能（Code Execution）も同画面で有効にしておいてください。

### ステップ 4：サンプルをプロジェクト知識に投入する

本レポジトリの [`knowledge-base-samples/`](./knowledge-base-samples/) 内の以下 4 ファイルを、プロジェクトのナレッジベースにアップロードします。

- `sample-en.txt` — 入力サンプル（英語版）
- `sample-jp.txt` — 入力サンプル（日本語訳版）
- `sample-output-dataset.md` — データセット出力の見本
- `sample-output-articles.md` — 記事出力の見本

これらは Claude が「何を作ればよいか」を学習するための教材です。

### ステップ 5：試運転

プロジェクト内で新しい会話を開始し、次のように話しかけます。

> こんにちは。プロジェクトナレッジのサンプル入出力を見て、この作業の型を把握してください。

Claude が型を把握したと応答すれば、セットアップは完了です。

## 毎週の運用手順

1. Telegram Desktop から WTM の英語チャネルと日本語訳チャネルを 1 週間分エクスポートします（形式：JSON）。ファイル名は `en.txt` と `jp.txt` にしておくと以降がスムーズです。
2. プロジェクト内で新しい会話を開き、2 つのファイルを添付して次のように依頼します。
   > 今週分の en.txt と jp.txt です。WTM News Digest の作業を開始してください。
3. Claude が投稿を抽出・分析し、統合データセットを生成します。
4. 続けて「メインニュース上位 5 トピックを選定してください」「各トピックの解説記事を執筆してください」「残りをジャンル別に整理してください」と順に依頼します。

各ステップの詳細な指示文は [`docs/how-it-works.md`](./docs/how-it-works.md) に記載しています。

## 出典と扱いの留意

本キットは、Telegram チャネル「WeTheMedia」および「WeTheMedia 日本語訳」の公開投稿を編集素材として扱うワークフローの複製キットです。サンプルデータはこれら公開チャネルからエクスポートしたものを使用しています。

WTM は保守色の強いチャネルであり、投稿の中にはヘイト表現や陰謀論的な内容が含まれる場合があります。本ワークフローは事実情報を正確に抽出しつつ、そうした投稿については「記事化不適」と要約欄で明示する設計になっています。編集者は最終出力に対して人間の判断を必ず加えてください。
